Sickwell Music Bot v4.7.5 Autopilot Update — Rules of the Game

By using the bot, you agree to these rules. If you don't agree — don't use it. Simple as a barrel of beer.

🏆 New in v4.7: Achievements and statistics system. By using the bot, you agree to storage of anonymized listening statistics (track count, time, genres) for achievements and leaderboards functionality.

1. What This Is

Sickwell_music is a Discord music bot with features: smart random with 7 rarity filters (micro/small/medium/large/mega/deep/any), hybrid commands (! and /), live queue with progress bar, 30 achievements system, autopilot mode and user statistics.

2. What You Cannot Do

  • Use the bot to violate Discord Terms of Service or local laws
  • Spam commands to cause malfunction (DDoS the bot)
  • Attempt to hack code or tokens (though they are in .env, but still)
  • Play NSFW content in inappropriate channels (server admin responsibility)
  • Intentionally exploit achievement statistics (bot detects this and can reset progress)

3. Music and Copyright

The bot uses YouTube as a source. We don't store music, don't distribute files, only stream. If your track appeared in rotation and you're unhappy — contact YouTube, not us.

Server admin is responsible for content that members play through the bot.

4. Statistics and Achievements

For the achievements system (v4.7+) the bot saves:

  • Number of played tracks and total listening time
  • Music genres (for achievements like "Phonker", "Rock Star")
  • First use date (for "Early Bird" achievement)
  • Server activity (for server leaderboard)

Data is stored in local SQLite database, not transferred to third parties, used only for displaying your statistics (/stats) and global top (/globaltop).

5. Availability

Bot works "as is". We don't guarantee 100% uptime (especially while on developer's PC, not Raspberry Pi yet). Donations go toward hardware, but this is not a subscription — no guarantees, only hope and coffee.

6. Terms Changes

We may update these rules. By continuing to use the bot after changes — you agree to them. Follow updates in the #📜╏updates channel.
