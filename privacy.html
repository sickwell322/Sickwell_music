Политика конфиденциальности

Последнее обновление: 10 февраля 2026

1. Какие данные собираем

  • ID сервера Discord (для работы очереди)
  • Имена пользователей (для отображения в очереди)
  • Запросы поиска музыки (временно, для воспроизведения)

2. Как используем

Данные используются только для функционирования бота. Не передаём третьим лицам.

3. Хранение

Данные хранятся только во время работы бота. История очищается при перезапуске.

4. Ваши права

Вы можете удалить бота с сервера в любой момент — все данные сервера будут удалены.
